Native vs. open-source

AI-in-One Dashboard vs. Viva Insights Copilot Dashboard

Microsoft ships a native Copilot dashboard inside Viva Insights. The same Microsoft team that helps run the largest Copilot deployments also ships an open-source extension called AI-in-One. Here's how they differ — and why most enterprises end up using both.

TL;DR

Viva Insights Copilot Dashboard is the fastest path if you already have a Viva license — zero deployment, refreshed automatically, focused on adoption and behavioural-change KPIs. AI-in-One is the open-source Power BI report you reach for when you need to combine Viva data with Purview audit logs, agent telemetry, sentiment surveys, or admin center exports — or when you simply don't have Viva and need the same answers. Both are built by the same Microsoft team.

What Viva Insights gives you out of the box

Viva Insights includes a Copilot dashboard that's been built and refined by Microsoft's behavioural-science team. It's the right starting point when:

  • You already pay for Viva Insights and want immediate visibility with no deployment effort.
  • The audience is line-of-business leadership and you need polished, prescribed KPIs rather than raw flexibility.
  • The questions are mostly "is adoption moving?" and "are behaviours changing?".

What AI-in-One adds

AI-in-One is open source and Power BI–based, so you own the data model. It's the right tool when:

  • You need to combine Viva data with Purview audit logs, agent telemetry, sentiment surveys, or admin center exports in a single report.
  • Finance or procurement is asking license-defence questions and you need cohort-level utilization rather than aggregate adoption.
  • You don't have Viva and need the same visibility from data you already own.
  • Your team wants to edit and rebrand the visuals or extend the model with internal data sources.

Use Viva Insights Copilot Dashboard if…

  • You already have a Viva license
  • You need answers today, with zero setup
  • Your audience prefers prescribed KPIs
  • Your questions stop at "is adoption moving?"

Use AI-in-One if…

  • You don't have Viva — or want to extend it
  • You need to combine multiple data sources
  • Finance wants cohort-level license defence
  • You want to own and customize the model

Most teams run both.

Viva for the at-a-glance executive view, AI-in-One for the deep dive your analysts and CFO ask for.
