      How tenant gating works

      These iframes use autoAuth=true with Microsoft's tenant ID (72f988bf-…). That means: visitors signed into the corp tenant see the report. Anyone else sees the sign-in screen and is denied. This page is also marked noindex,nofollow so it won't appear in Google.
