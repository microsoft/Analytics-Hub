Where do I get these files?

Each browser app above runs on two CSV exports you already own — nothing is uploaded, everything is parsed locally in your browser:

  1. Entra user export(s) — in the Microsoft Entra admin center, open Identity → Users → All users and choose Download users. This CSV carries the org attributes (Department, Cost Center, Job title, etc.) used to allocate cost and group people.
  2. Copilot credit / Cost Management export — export your Copilot Cowork consumption from the Microsoft admin center under Cowork Cost Management, or use the credit / usage CSV from your Microsoft billing contact. It just needs one row per user with the credits they consumed.

Column names are auto-detected, so most export layouts work as-is. The Power BI report & template uses the same two sources.
