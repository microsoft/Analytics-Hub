Glossary

Every Copilot reporting term — in plain English.

If you've ever had to translate "depth of use" or "super user cohort" for an exec, this is for you. Bookmark it, send it to your team, and we'll keep it current as Copilot evolves.

0 of 0 terms

Adoption #

The percentage of licensed users who are actively using Copilot in a given period, typically measured weekly or monthly. The headline number on most executive Copilot dashboards.

Agent (Copilot Agent) #

A purpose-built Copilot extension — declarative, custom engine, or Studio-authored — that responds to a defined scope of prompts.

Agent Telemetry #

Usage data emitted by Copilot agents — invocations, authors, departments, outcomes — surfaced in the Agent Intelligence Report.

AI-in-One Dashboard #

The flagship executive Power BI report in the Analytics Hub that consolidates adoption, usage, depth, and sentiment into a single view.

Audit Reader #

A Microsoft 365 role that grants permission to search the Purview unified audit log. A prerequisite for running PAX.

BVA (Business Value Advisor) #

A Microsoft role focused on quantifying and articulating the business value of a customer's Microsoft investment, especially around Copilot ROI conversations.

Champion ID #

A workflow in the CustomizeCopilot toolkit that helps identify and onboard internal Copilot champions.

Chat Intelligence Report #

A Power BI template in the CopilotChatAnalytics repo that surfaces prompt-level chat patterns from Purview audit data.

Cohort #

A grouping of users by behaviour — active, recently active, dormant, super user — used to compare engagement across populations.

Copilot Adoption & Sentiment Report #

A Power BI template that combines adoption telemetry with survey-based sentiment scores to explain why adoption is or isn't moving.

CSM (Customer Success Manager) #

A Microsoft role that owns the customer's outcomes against their Copilot deployment.

Custom Engine Agent #

A Copilot agent built on a custom orchestration layer rather than the declarative format. Surfaced separately in the Agent Intelligence Report because its telemetry shape differs from declarative agents.

Decoding Super Usage #

A Power BI template that breaks down what super users do differently — surfaces, prompt patterns, depth — so the behaviour can be replicated.

Declarative Agent #

A YAML-defined Copilot agent that runs on the standard Copilot orchestration. The most common agent type in enterprise deployments.

Depth of Use #

How many Copilot surfaces (Word, Excel, Teams, Outlook, Loop, etc.) an individual actively uses. A leading indicator of long-term retention — deeper users churn less.

DirectQuery #

A Power BI connection mode where queries run live against the source rather than against a cached import. Used by the DirectQuery variants of Decoding Super Usage for tenants that need near-real-time refresh.

Dormant License #

An assigned Copilot license that has shown no measurable usage over a defined period. A primary candidate for reclamation in license-defence conversations.

License Utilization #

Ratio of active users to assigned licenses for Microsoft 365 Copilot. The core defensibility metric for license renewal conversations with finance and procurement.

M365UsageAnalytics #

A Microsoft 365 Usage Analytics scaffold in the Analytics Hub that connects native M365 usage data to the broader Copilot reporting model.

PAX #

PowerShell Audit Exporter — the Analytics Hub CLI that pulls Copilot interactions from Purview Audit and writes them to CSV for the Chat and Agent Intelligence reports. Replaces manual Search-UnifiedAuditLog calls.

PBIT #

A Power BI Template file. The .pbit format used by every Power BI report in the Analytics Hub — opens in Power BI Desktop, prompts for parameters, then loads data.

People Scientist #

A Microsoft role that applies behavioural science to enterprise Copilot adoption — focused on the human side of measurement.

Prompt Depth #

A measure of how substantive a user's prompts are — length, follow-up turns, surface variety — surfaced in the Chat Intelligence Report.

Purview Audit #

Microsoft's unified audit log service. Records Copilot interactions and serves as the data source for PAX, which feeds Chat and Agent Intelligence reports.

Sentiment Score #

A composite score derived from survey responses about Copilot, scaled 0–100. Used in the Copilot Adoption & Sentiment Report to explain adoption movement.

Super User #

A licensed user whose Copilot behaviour places them in the top cohort by depth, frequency, and surface breadth. The population whose practices the Super User Impact and Decoding Super Usage reports model and propagate.

Super User Impact #

A Power BI template that quantifies the marginal value attributable to the super user cohort. The most direct artifact for defending license waves in front of finance.

Viva Insights Copilot Dashboard #

Microsoft's native dashboard inside Viva Insights for Copilot adoption, behavioural change, and ROI. Extended by several Analytics Hub reports — see the comparison page.
